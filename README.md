# 简介

自用，暂无开关。  
稍微改进哔哩哔哩视频及直播播放器外观。  
浏览器安装油猴插件后点击[这里](https://github.com/xmfm/bilibili_player_looks/raw/master/bilibili_player_looks.user.js)安装脚本。  

- 视频播放器
    - 隐藏画面右上角关注和反馈键
    - 隐藏暂停时画面上的暂停按钮
    - 改变弹幕开关键样式
    - 弹幕发送键只在弹幕文本框有焦点或文字时有颜色
- 直播播放器
    - 固定顶栏
    - 隐藏2233娘
    - 隐藏画面左上的logo
    - 隐藏弹幕栏底部的免费礼物栏
    - 隐藏头衔
